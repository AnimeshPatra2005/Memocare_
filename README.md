# MemoCare 🧠

MemoCare is an end-to-end encrypted medical screening assistant. It utilizes deep learning (DenseNet201) for MRI brain scan classification and tabular ensemble classifiers for cognitive assessment screening.

All patient metrics and verdicts are encrypted client-side in the browser before being stored in the database.
